# PRD — 양도소득세 신고서 작성 웹앱(로컬/데스크탑)

## 1. 문제 정의
- 사용자는 2023년 이후 양도한 자산에 대해 **양도소득세 신고서(서식/부표/첨부서류)** 를 작성해야 한다.
- 홈택스는 기능이 방대하고, 입력-계산-출력 흐름이 복잡해 **실무형 문서 작성 도구**가 필요하다.

## 2. 목표
- 사용자가 사실관계를 입력하고(판정은 사용자가 결정), 앱이 법령/서식에 따라:
  - 계산,
  - 검증,
  - 부표 자동연계,
  - 신고서 PDF 출력  
  까지 **정확하게** 수행한다.

## 3. 범위/제약
- 거주자만 지원
- 로컬 실행(브라우저/데스크탑), 업로드/서버 저장 없음
- 2023년 이후 양도분
- 출력까지만 지원(납부서 등 미지원)
- 한국어만

## 4. 핵심 플로우(스텝퍼)
1) 케이스 생성(신고구분/과세연도/신고일)
2) 신고인(거주자) 인적사항 입력(로컬 저장)
3) 자산 추가(부동산/권리/기타/주식/파생/신탁 등)
4) 취득/양도 금액 및 필요경비(부표3 연계)
5) 세율구분/특수규정/감면(사용자 판정값 입력 + 계산)
6) 합산/가산세/공제(자동)
7) 미리보기(전체 서식 페이지)
8) 다운로드(PDF) / 단면 인쇄

## 5. 성공 기준(완전성)
- Field Catalog Gate=Y 100% 구현
- Rule Matrix Gate=Y 100% 구현
- Scenario Test Suite 대표세트 100% 통과
- 출력 PDF가 서식 구성과 일치(좌표/서체 차이는 허용, 필드/페이지 구성은 일치)
