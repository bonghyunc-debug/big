# Scenario Test Suite (대표 + 경계값)

목표: “홈택스 일반신고” 수준의 계산·검증·출력 경로를 최소 대표세트로 강제 커버.

## 1) 시나리오 파일 규격
- 위치: `tests/scenarios/<ScenarioID>.json`
- 구조:
  - `case`: 입력(사용자 판정값 포함)
  - `expect`: 기대 결과(중간값 포함; 원단위)
  - `evidence`: 근거 링크/메모
  - `notes`

## 2) 최소 대표세트(필수)
아래는 최소 대표세트(각 1개 + 경계 2개 권장).

### S-REAL-001 토지(일반세율) + 장특공(표1)
- 보유 3년/15년 경계 포함

### S-REAL-002 주택(1세대1주택 비과세, 고가주택 12억 초과 안분)
- 12억 exactly / 12억 초과 경계 포함

### S-REAL-003 분양권/조합원입주권(단기세율)
- 1년 미만 / 1~2년 / 2년 이상 경계 포함

### S-REAL-004 비사업용토지(중과) + 장특공 배제/제한
- 비사업용 판정 입력값 반영

### S-REAL-005 미등기 양도(단일세율) + 기본공제 제외
- 기본공제 0 확인

### S-STOCK-001 주식(상장/비상장, 코드별)
- 세율코드 1-60/1-61 등은 룰팩에 따라 대표 케이스 선정

### S-DERIV-001 파생상품(세율구분코드 81)
- 산식(⑧-⑨-⑪-⑫+⑬-⑭) 그대로 검증

### S-PEN-001 기한후신고(가산세 + 감면율)
- 국세기본법 제48조 감면율 경계 포함

### S-PEN-002 수정신고(과소신고 + 납부지연 + 감면)
- 납부지연일수 0/1/30 경계 포함

### S-GIFT-001 부담부증여(시행령 159조 안분)
- 채무비율 0%/50%/100% 경계 포함

### S-CARRY-001 이월과세(소득세법 97조의2)
- 10년/1년 경계 포함, 증여세 상당액 필요경비 산입 확인

## 3) 골든 비교(권장)
- 사용자가 홈택스에서 동일 입력 → 출력(PDF) 저장
- 우리 앱 출력(PDF)과 필드 값/페이지 구성 비교
