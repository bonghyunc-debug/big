# 별지 제84호 스키마/서식 개정 관리 로그 템플릿

## 기록 절차
1. 개정 사실 확인 후 아래 템플릿을 복사하여 새 섹션을 추가한다.
2. 법령 개정·서식 개정·세율 변경 등 영향 범위를 명시하고, 관련 스키마/테이블(예: `schemas/2024/form.json`, `schemas/2024/schedules/schedule1.json`)의 수정 내역을 bullet로 정리한다.
3. 서식 교체 시 `assets/forms/{year}/`에 파일을 저장하고 `manifest.json`의 SHA를 갱신한다.
4. 변경 후 lint/type/test 명령을 실행하고 결과를 로그 하단에 기록한다.
5. PR 제출 시 본 로그에서 해당 개정 항목을 참조하도록 링크한다.
