# 자산 유형·감면 유형별 첨부서류 매트릭스

본표 및 부표 입력 필드와 첨부서류를 매핑해 자동 출력할 수 있도록 정리하였다. 출력 순서는 동일 자산·감면 유형 내에서 상단부터 하단 순으로 반영한다.

| 자산 유형 | 감면 유형 | 본표 필드/부표 필드 ↔ 첨부서류 매핑·출력 순서 | 법령·서식 작성방법 근거 |
| --- | --- | --- | --- |
| 토지 | 비과세 (예: 1세대 1주택 부수토지) | 1. 본표 `자산종류`, `취득일`, `양도일` → **등기사항증명서**(취득·양도 시점 확인)  <br>2. 부표 `면적`, `지목`, `용도지역` → **토지(임야)대장**  <br>3. 부표 `취득가액 산정` → **매매계약서·공인중개사 확인서** | 소득세법 §89, 소득세법 시행규칙 별지 제84호 서식 작성안내(토지·건물 양도소득과세표준 확정신고서) |
| 토지 | 장기보유특별공제 | 1. 본표 `보유기간` → **취득·양도 등기사항증명서**(보유기간 증빙)  <br>2. 부표 `토지 양도가액 계산` → **감정평가서**(시가 산정 시)  <br>3. 부표 `보유기간 특례` → **무허가건축물 확인서**(필요 시) | 소득세법 §95·§104의3, 시행규칙 별지 제84호 서식 작성방법(장기보유특별공제 입력란) |
| 토지 | 조세특례 감면 (예: 공익사업 수용) | 1. 본표 `감면사유 코드` → **수용통지서**  <br>2. 부표 `보상가액` → **보상평가서**  <br>3. 부표 `감면세액 계산` → **조세특례제한법 감면신청서** | 조세특례제한법 §77·§121, 조특법 시행규칙 별지 감면신청서 서식 작성안내 |
| 건물 | 비과세 (주택, 1세대 1주택 요건) | 1. 본표 `주택유형`, `세대구성` → **주민등록등본**  <br>2. 본표 `실거주기간` → **전입세대열람내역**  <br>3. 부표 `건물면적` → **건축물대장**  <br>4. 부표 `취득가액` → **매매계약서/분양계약서** | 소득세법 §89, 시행규칙 별지 제84호 서식 작성안내(1세대1주택 비과세) |
| 건물 | 장기보유특별공제 | 1. 본표 `취득일`, `양도일` → **등기사항증명서**  <br>2. 부표 `필요경비` → **리모델링·수선비 영수증**  <br>3. 부표 `실지거래가액` → **매매계약서** | 소득세법 §95, 시행규칙 별지 제84호 서식 작성방법(장기보유특별공제) |
| 건물 | 조세특례 감면 (임대주택 등) | 1. 본표 `감면대상 주택등록번호` → **민간임대주택 등록증**  <br>2. 부표 `임대기간·임대료` → **임대차계약서**  <br>3. 부표 `감면세액 계산` → **조세특례제한법상 감면신청서** | 조세특례제한법 §97의3, 시행규칙 임대주택 감면신청서 작성안내 |
| 분양권 | 비과세 (주택 전환 후) | 1. 본표 `주택전환 여부` → **사용검사필증/준공검사필증**  <br>2. 부표 `분양대금 납부내역` → **분양대금 납부영수증**  <br>3. 부표 `계약변경` → **분양계약서 및 변경계약서** | 소득세법 §89, 시행규칙 별지 제84호 서식 작성안내(분양권 전환 주택) |
| 분양권 | 장기보유특별공제 | 1. 본표 `계약체결일`, `전매일` → **분양계약서**  <br>2. 부표 `중도금 납부일자` → **납부확인서**  <br>3. 부표 `보유기간 산정` → **등기예정확인서(있을 경우)** | 소득세법 §95, 시행규칙 별지 제84호 서식 작성방법(분양권 보유기간 산정) |
| 분양권 | 조세특례 감면 (신축주택 취득세 감면 연계) | 1. 본표 `감면코드` → **지방세 감면 확인서**  <br>2. 부표 `취득가액` → **분양계약서**  <br>3. 부표 `감면세액 계산` → **조세특례제한법 감면신청서** | 조세특례제한법 §99의3 등, 시행규칙 감면신청서 작성안내 |
| 조합원입주권 | 비과세 (재개발·재건축 1세대1주택 특례) | 1. 본표 `조합원번호`, `관리처분인가일` → **관리처분계획인가서**  <br>2. 본표 `원주택 보유기간` → **구 주택 등기사항증명서**  <br>3. 부표 `분담금` → **분담금 납부영수증** | 소득세법 §89, 시행령 §154, 시행규칙 작성안내(조합원입주권) |
| 조합원입주권 | 장기보유특별공제 | 1. 본표 `입주권 취득일`, `양도일` → **조합 가입계약서**  <br>2. 부표 `건축기간 특례 보유기간` → **사업시행인가서**  <br>3. 부표 `취득가액` → **분담금 납부내역** | 소득세법 §95, 시행령 §162의3, 시행규칙 작성방법(입주권 보유기간 특례) |
| 조합원입주권 | 조세특례 감면 (이주대책 특별분양 등) | 1. 본표 `감면사유` → **이주대책 특별분양 확인서**  <br>2. 부표 `보상가액` → **보상계약서**  <br>3. 부표 `감면세액 계산` → **조세특례제한법 감면신청서** | 조세특례제한법 §77·§99의4, 시행규칙 감면신청서 작성안내 |
| 비상장주식 | 비과세 (창업벤처 중소기업 주식 양도소득) | 1. 본표 `주식종류`, `지분율` → **주주명부/주식등변동상황명세서**  <br>2. 부표 `취득가액`, `양도가액` → **주식양도계약서**  <br>3. 부표 `회사 요건` → **중소기업확인서·벤처기업확인서** | 소득세법 §94·§97, 시행령 §157, 시행규칙 별지 제84호 서식 작성안내(주식 양도) |
| 비상장주식 | 장기보유특별공제 (사업관련 주식) | 1. 본표 `보유기간` → **주식취득일 증빙(주식인수계약서)**  <br>2. 부표 `시가 산정` → **비상장주식 평가명세서**  <br>3. 부표 `필요경비` → **취득·관리비 영수증** | 소득세법 §95, 시행령 §157의4, 서식 작성방법(비상장주식 평가) |
| 비상장주식 | 조세특례 감면 (중소기업 통합·사업전환 특례) | 1. 본표 `감면코드` → **조세특례제한법 감면신청서**  <br>2. 부표 `합병·분할 계약` → **합병(분할)계약서**  <br>3. 부표 `주식평가` → **합병·분할 기준일 재무제표** | 조세특례제한법 §31·§31의2, 시행규칙 감면신청서 작성안내 |
